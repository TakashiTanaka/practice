画面幅 - 左右のpadding = 要素幅

このBOXは親要素のpadding分、幅が狭まる

画面幅 - 左右のpadding + ネガティブマージン = 要素幅

このBOXはネガティブマージンを適用しているので、幅が画面幅になる

画面幅800px以下 → 画面幅 - 左右のpadding = 要素幅
画面幅800px以上 → 800px = 要素幅

このBOXはmax-width: 800px;が適用されているので、
画面幅が800px以下だと親要素のpadding分、幅が狭まるが、
画面幅が800px以上になると幅が800pxに固定される
