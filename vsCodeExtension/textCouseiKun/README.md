# テキスト校正くん

## テキスト校正くんとは

株式会社ICSが開発したVS Codeの拡張機能。
VSCodeでテキストファイル、Markdownファイルの日本語の文章をチェックしてくれる。
編集時に自動で校正のチェックを行い、問題パネル指摘内容を表示してくれる。
校正ルールを設定で変更することもできる。
