# yarnの基本的な使い方

基本的にnpmを使っているけど、yarnを全く使ったことがないというものもアレなので試してみる

## yarnをグローバルインストール

`npm i -g yarn`

## バージョン確認

`yarn -v`

## ディレクトリを作ってinit

`yarn init -y`

## パッケージをインストール

`yarn add lodash react redux`

これで`node_modules`と`yarn.lock`ができる
