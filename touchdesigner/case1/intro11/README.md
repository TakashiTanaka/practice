# Noise CHOPでオブジェクトを作る！

Noise CHOPのChannelタブ内、Channel Namesにほしい数のチャンネルを書く

`tx ty tz`

正規表現を使って、下記のようにもかける

`t[xyz]`

PalletにMy Conponents入れたい。

色が暗い。。
