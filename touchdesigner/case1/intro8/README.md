# インスタンシングとカメラワーク

インスタンシングとは。。
GPUを使ってリアルタイムに3Dオブジェクトをたくさんコピーできる。ジオメトリインスタンスともいう。

SOP to CHOP
SOPの情報をCHOPに変換

info CHOP
オペレーターの情報を見れる

TOP to CHOP
TOPの情報をCHOPに変換

Merge CHOP
情報をまとめる

Merge SOP
SOPをまとめる
